# Amelu Picosas

Sitio web de una sola página para Amelu Picosas — salsas y encurtidos picantes hechos a mano en lotes pequeños (Jalea de Ají Charapita, Mayonesa de Jalapeños, Jalapeños Encurtidos y Aceite de Ají Charapita).

Sitio estático (HTML/CSS puro). Para verlo localmente, abre `index.html` en el navegador.

## Despliegue en Vercel
Sitio estático sin build. En Vercel: **Add New → Project → Import** este repositorio (Framework Preset: *Other*, sin build command) y desplegar.
